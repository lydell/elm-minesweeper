# elm-minesweeper

The classic game MineSweeper made with Elm. **[Play it.][play]**

## Development

1. `npm install`
2. `npm run build:css`
3. `npm start`

Add `?debug=1` to the URL to run in debug mode.

`npm run analyse` runs [elm-analyse].
